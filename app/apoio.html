Passos:

  1. Crie um Novo Cenário

    Clique em "Create a new scenario" para iniciar um novo cenário.

  2. Adicione o Módulo Webhook

    Procure por Webhooks e selecione Custom Webhook. Clique em "Add".

  3. Crie um Novo Webhook

    No módulo do Webhook, clique em "Create a webhook". Copie o URL gerado (esse será o link usado no JavaScript).

  4. Configure o Webhook para JSON

    Em "Settings", escolha JSON como o formato de entrada e salve as configurações.

  5. Conecte ao Google Sheets

    Adicione um novo módulo ao cenário e selecione Google Sheets. Escolha a ação Add a row to a sheet.

  6. Mapeie os Dados

    Conecte sua conta do Google Sheets e selecione a planilha e a aba onde os dados serão inseridos. Mapeie os campos recebidos do Webhook para as colunas da planilha.

    • studentName → Coluna para o nome do aluno.
    • vocabulario → Coluna para o vocabulário.
    • pronuncia → Coluna para a pronúncia.
    • gramatica → Coluna para a gramática.
    • comentariosGerais → Coluna para comentários gerais.
    • comentarioInterno → Coluna para comentário interno.
    • fpe, cor, pat, q → Colunas para as notas.
  7. Teste o Webhook

    Clique em "Run once" para testar o cenário. Use uma ferramenta como Postman ou cURL para enviar um exemplo de JSON para o Webhook.

    
curl -X POST https://hook.us2.make.com/SEU_WEBHOOK_URL \
-H "Content-Type: application/json" \
-d '{
  "studentName": "João Silva",
  "vocabulario": "Bom uso de palavras.",
  "pronuncia": "Pronúncia clara.",
  "gramatica": "Boa estrutura gramatical.",
  "comentariosGerais": "Excelente desempenho.",
  "comentarioInterno": "Aluno dedicado.",
  "fpe": 3.5,
  "cor": 2.0,
  "pat": 1.5,
  "q": 1.0
}'
  8. Atualize o Campo `studentName` Dinamicamente (Opcional)

    Para preencher automaticamente o campo studentName, crie outro Webhook para consultar a planilha e retornar o nome do aluno. Configure o JavaScript para fazer uma chamada GET ao Webhook de consulta.
